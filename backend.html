백엔드란?

백엔드는 사용자가 직접 볼 수 없는 서버, 데이터베이스, API 등
웹사이트와 앱의 핵심 로직과 데이터를 처리하는 영역입니다.
Node.js, Python, Java, PHP 등 다양한 언어와 프레임워크로
데이터 저장, 인증, 비즈니스 로직, API 제공 등을 담당합니다.

백엔드 기초 예제 (Node.js Express)
// Node.js + Express로 만드는 간단한 서버 예제
const express = require('express');
const app = express();
const PORT = 3000;

app.get('/', (req, res) => {
  res.send('Hello Backend!');
});

app.listen(PORT, () => {
  console.log(`서버가 http://localhost:${PORT} 에서 실행 중입니다.`);
});
