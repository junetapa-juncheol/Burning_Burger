프론트엔드란?

프론트엔드는 사용자가 직접 보고, 만지고, 경험하는 웹의 모든 UI/UX를 만드는 영역입니다.
HTML, CSS, JavaScript를 기반으로 다양한 프레임워크(React, Vue, Angular 등)와 도구를 활용해
웹사이트와 웹앱의 인터페이스를 구현합니다.

프론트엔드 기초 예제 (HTML+CSS+JS)
<!DOCTYPE html>
<html lang="ko">
<head>
  <meta charset="UTF-8">
  <title>Hello Frontend!</title>
  <style>
    body { background: #f5f7fa; font-family: sans-serif; }
    h1 { color: #4a3aff; }
    button { padding: 10px 20px; border-radius: 8px; border: none; background: #4a3aff; color: #fff; cursor: pointer; }
    button:hover { background: #3730a3; }
  </style>
</head>
<body>
  <h1>프론트엔드 시작!</h1>
  <button onclick="alert('프론트엔드 세계에 오신 것을 환영합니다!')">클릭해보세요</button>
</body>
</html>
